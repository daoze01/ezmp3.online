# EzMP3 - 免费YouTube视频转MP3工具

## 项目概述

EzMP3是一个功能强大的在线视频转MP3工具，可以从YouTube、Vimeo等平台提取音频并下载为MP3格式。该项目采用前后端分离架构，前端使用HTML、CSS和JavaScript构建，后端使用Node.js提供API服务。

## 主要功能

- **视频转MP3**: 从YouTube、Vimeo、Facebook等平台转换视频到MP3格式
- **视频信息预览**: 粘贴URL后自动获取视频标题、缩略图和时长
- **音频剪辑**: 支持设置开始和结束时间，仅转换需要的部分
- **多种音质选择**: 支持64k、128k、192k和320k比特率
- **下载历史记录**: 自动保存转换历史，方便重复下载
- **黑暗模式**: 支持浅色/深色主题切换，减少夜间使用的眼睛疲劳
- **多语言支持**: 支持英文和中文界面，易于扩展更多语言
- **响应式设计**: 完美适配PC、平板和手机等各种设备

## 第二阶段改进

相比第一阶段的静态模拟版本，第二阶段实现了实际功能：

1. **实际API集成**: 添加了后端服务，实现真实的视频信息获取和MP3转换
2. **视频信息预览**: 粘贴URL后立即显示视频信息，提升用户体验
3. **音频剪辑功能**: 支持选择视频的特定部分进行转换
4. **黑暗模式**: 实现了深色主题，支持根据用户偏好自动切换
5. **多语言支持**: 添加中英文语言切换功能
6. **改进的UI交互**: 更细致的转换状态反馈和下载通知
7. **代码重构**: 更模块化的代码结构，便于未来扩展

## 技术栈

### 前端
- HTML5
- CSS3 / Tailwind CSS
- JavaScript (ES6+)
- 响应式设计

### 后端
- Node.js
- Express.js
- ytdl-core (YouTube下载库)
- fluent-ffmpeg (音频处理)

## 快速开始

### 前端

直接在浏览器中打开 `index.html` 文件即可预览UI。

### 后端

1. 进入后端目录：
   ```
   cd server
   ```

2. 安装依赖：
   ```
   npm install
   ```

3. 创建环境变量文件：
   ```
   cp .env.example .env
   ```

4. 启动服务器：
   ```
   npm start
   ```

5. 服务器将运行在 http://localhost:3000
